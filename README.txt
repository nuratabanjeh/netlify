To use this demo you need to do the following steps:

#1: make sure that installing turtlebot3 & ros bridge:
bash turtlebot3_setup.sh

#2: run the turtlebot3 simulation & ros bridge:
bash urtlebot3_run.sh

#3: In New Terminal: goto inside "my-app" folder, and make sure that installing all packages:
npm install --force

#4: start the react app:
npm start

#Note: When you start the application 'npm start' an error message may appear: 'webpack compiled with 1 error' Because some bugs in npm/react or installed packages version. 
keep trying to run 'npm start' every 20 seconds until the application starts without error messages.
